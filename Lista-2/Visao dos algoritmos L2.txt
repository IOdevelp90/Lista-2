Exercício 1:
Este exercício pede um programa que leia um vetor de 5 números digitados pelo usuário, calcule a média desses números e então apresente na tela quais valores são menores, iguais ou superiores à média.

Exercício 2:
Neste exercício, o programa deve ler um vetor de números digitados pelo usuário e, em seguida, mostrar o número de cada posição do vetor e indicar se ele é positivo, negativo ou zero.

Exercício 3:
O objetivo deste exercício é criar um programa que leia um vetor de números digitados pelo usuário, crie um segundo vetor que armazene o dobro de cada número do primeiro vetor e depois apresente os valores deste segundo vetor.

Exercício 4:
Neste exercício, o programa precisa ler um vetor de 5 números digitados pelo usuário e uma variável de um número qualquer, e então mostrar na tela os índices dos elementos do vetor que são inferiores ao número informado.

Exercício 5:
O programa deste exercício deve ler 5 números do teclado e armazená-los em um vetor. Em seguida, deve ler um número n qualquer e apresentar na tela a quantidade de vezes que o número n aparece no vetor.

Exercício 6:
Este exercício pede um programa que leia um vetor com 5 notas de um aluno e um outro vetor com 5 pesos, e então calcule a média ponderada do aluno utilizando as notas e pesos fornecidos.

Exercício 7:
Neste exercício, o programa deve ler dois vetores de 5 elementos cada e verificar se eles são iguais ou não. Para serem iguais, todos os elementos dos dois vetores devem coincidir.

Exercício 8:
O objetivo deste exercício é criar um programa que dados dois vetores A e B de 5 elementos cada, crie um vetor C que represente a concatenação de A e B, ou seja, C contém os elementos de A seguidos dos elementos de B.

Exercício 9:
Neste exercício, o programa deve ler os vetores A e B de 5 elementos e gerar um vetor C de acordo com as seguintes regras: os elementos das posições pares de C são os elementos das posições pares de A, e os elementos das posições ímpares de C são os elementos das posições ímpares de B.

Exercício 10:
O programa deste exercício deve ler um vetor A de 5 posições e, ao final da leitura, copiar os elementos de A em um vetor B de forma invertida.

Exercício 11:
Este exercício pede um programa que calcule o produto escalar entre dois vetores de inteiros de tamanho igual a 5.

Exercício 12:
Neste exercício, o programa deve ler um vetor de 12 elementos e apresentar na tela o produto dos elementos pares positivos, desconsiderando o zero.

Exercício 13:
O objetivo deste exercício é criar um programa que receba um número n e retorne um vetor com os n primeiros termos da sequência de Fibonacci.

Exercício 14:
Este exercício pede um programa que leia um vetor de 5 elementos e, após a leitura, posicione o maior elemento na última posição do vetor, sem apagar ou duplicar nenhum número.

Exercício 15:
Neste exercício, o programa deve ler 5 números do teclado e preencher um vetor de acordo com a seguinte regra: com exceção do 1º número, só é permitido armazenar um número se ele for maior que o anterior.
